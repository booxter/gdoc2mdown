============
Installation
============


From sources
------------

The sources for Gdoc2Mdown can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/booxter/gdoc2mdown

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/booxter/gdoc2mdown/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/booxter/gdoc2mdown
.. _tarball: https://github.com/booxter/gdoc2mdown/tarball/master
